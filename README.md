# Parallelini Convoluchini

## Описание

Программа применяет различные фильтры свертки к изображениям с поддержкой многопоточной обработки. Реализованы несколько вариантов параллельной обработки изображений.

## Сборка программы

Для сборки:

```bash
make
```
Будет создан исполняемый файл convolution.out.

## Использование

```bash
./convolution.out [ОПЦИИ] входное_изображение выходное_изображение
```

### Основные опции
Опция	Описание	Значение по умолчанию
--filter=ТИП	Тип применяемого фильтра (обязательно)	-
--mode=РЕЖИМ	Режим обработки (обязательно)	-
--threads=ЧИСЛО	Количество потоков	(default:4)
--block=РАЗМЕР	Размер блока для режима by_grid	(default:8)
--help	Вывод справки	-
#### Доступные фильтры
1. blur - Размытие

2. soft_blur - Мягкое размытие

3. gaus_small_blur - Гауссово размытие (малое)

4. sharpen - Увеличение резкости

5. motion - Размытие в движении

6. gaus_blur - Гауссово размытие

7. negative - Негатив

8. random - Случайный фильтр

#### Режимы обработки
1. seq - Последовательная обработка

2. by_row - Параллельно по строкам

3. by_column - Параллельно по столбцам

4. by_pixel - Параллельно по пикселям

5. by_grid - Параллельно блоками

#### Примеры использования
Простое размытие (последовательно):

```bash
./convolution.out --filter=blur --mode=seq input.bmp output.bmp
```

Увеличение резкости параллельно в 8 потоков по строкам:

```bash
./convolution.out --filter=sharpen --mode=by_row --threads=8 input.png result.png
```
Размытие блоками 64x64:

```bash
./convolution.out --filter=motion --mode=by_grid --block=64 photo.bmp processed.bmp
```
